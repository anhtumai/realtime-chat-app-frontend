# Realtime chat app frontend

Realtime chat app based on WebSocket backend

## Usage

Requirement:

- Node package manager: Yarn or Npm

### Install dependencies

```bash
yarn install
```

### Run on localhost

```bash
# default value for REACT_APP_WS_URL is
REACT_APP_WS_URL="" yarn run start
```

### Build

```bash
yarn run build

# then

npx serve -s build
```
